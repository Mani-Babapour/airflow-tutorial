# airflow-tutorial

This repository contains my hands-on practice with Apache Airflow using Docker.

## Overview

The goal of this project is to understand how Apache Airflow works in a containerized environment and how to build and run simple workflows (DAGs).

Key topics covered:
- Basics of Apache Airflow
- DAG (Directed Acyclic Graph) concepts
- Task dependencies and scheduling
- Running Airflow using Docker Compose
- Writing and testing simple DAGs
